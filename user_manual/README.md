# User Manual

This directory contains the user documentation for the AquaSol Smart Irrigation System. It provides step-by-step instructions for installation, system setup, operation, and troubleshooting.

## Contents

- **Installation Guide** – Hardware assembly and initial setup.
- **Configuration Guide** – ESP32, LoRa, Wi-Fi, and sensor configuration.
- **Operating Instructions** – Using the web dashboard and mobile application.
- **Maintenance Guide** – Routine maintenance and system checks.
- **Troubleshooting** – Common issues and recommended solutions.

This documentation is intended to help users deploy, configure, operate, and maintain the AquaSol system efficiently.
