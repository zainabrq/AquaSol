# Schematics

This folder contains the hardware design and system architecture schematics for the AquaSol project.

## Included

- **Master Node Schematic** – ESP32 gateway responsible for LoRa communication, sensor data aggregation, cloud connectivity, and irrigation control.
- **Node 1 Schematic** – ESP32-based sensor node for monitoring soil moisture, temperature, humidity, and environmental parameters.
- **Node 2 Schematic** – ESP32-based sensor node for distributed field sensing and wireless LoRa communication.

These schematics illustrate the electrical connections, peripheral interfaces, and communication architecture of the AquaSol embedded system.
